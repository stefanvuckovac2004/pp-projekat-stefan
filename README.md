Opis:
Program implementira rad sa dvodimenzionalnim kvadratnim matricama dimenzije m × m.
Omogućene su sljedeće funkcionalnosti:

unos dvije matrice
ispis unesenih matrica

Program je izrađen korištenjem proceduralnog pristupa, uz posebne funkcije za svaku od operacija. Implementirane su i potrebne validacije korisničkog unosa.
